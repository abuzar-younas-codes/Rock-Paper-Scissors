# 🎮 Rock Paper Scissors Game

This is my first JavaScript game, built while learning the basics of HTML, CSS, and JS. It’s a classic game with a clean and interactive interface.

<img width="1596" height="771" alt="app-screenshot" src="https://github.com/user-attachments/assets/f42dffdb-8472-41fa-8f21-f73db1e21ee5" />

## ✨ Features
- **User vs Computer:** Play against an automated computer opponent.
- **Score Tracking:** Real-time score updates to keep track of wins and losses.
- **Interactive UI:** Smooth animations and a user-friendly design.

## 🛠️ Tech Stack
- **HTML5:** For the game structure.
- **CSS3:** For styling and animations.
- **JavaScript:** For the game logic and score management.
